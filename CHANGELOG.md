# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- `py.typed` marker so type information ships to consumers (PEP 561).
- pytest suite covering configuration parsing and the gateway routes.
- GitHub Actions CI (lint, test, build, `twine check`) and a docs deploy workflow.
- MkDocs Material documentation site with an autogenerated API reference.

### Changed
- Migrated the gateway from the deprecated `@app.on_event("shutdown")` to the
  FastAPI lifespan context manager.
- Packaging metadata expanded for PyPI (classifiers, keywords, project URLs,
  SPDX license, dynamic single-sourced version).

## [0.1.0]

### Added
- Initial release: an OpenAI-compatible gateway that proxies
  `/v1/chat/completions`, `/v1/models`, and `/health` to a user-supplied upstream
  (llama.cpp, vLLM), with streaming support and request sanitization.
- `wavecat-sdk serve` CLI and `WAVECAT_SDK_*` environment configuration.
